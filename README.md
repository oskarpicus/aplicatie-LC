# Conversii si operatii in baze de numeratie
Aplicatie cu interfata de tip consola - suporta conversia intre doua numere naturale in baze de numeratie diferite de 10 si operatii aritmetice in bazele 2,3,...,10,16
<ins>Algoritmi utilizati</ins>
Conversii intre baze | Operatii aritmetice
-------------------- | -------------------
Împărțiri succesive | Adunare
Utilizarea unei baze intermediare | Scădere
Substituție | Înmulțire cu o cifră
Conversii rapide între baze puteri ale lui 2 | Împărțire la o cifră
